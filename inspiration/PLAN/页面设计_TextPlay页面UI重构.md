# 页面设计：Text play（文本游玩）页面 UI 重构

## 1) Layout（布局）
- 主布局：`TopBar + Content` 的纵向堆叠结构（Flex Column）。
- Content 区：主内容占满可用空间；浮动面板使用 `position: fixed`（或 `absolute` + portal 到 body）覆盖在内容之上，不挤压主内容。
- 间距与对齐：统一使用 8px 栅格（8/16/24）；TopBar 高度固定；面板与视口边缘保持 16px 安全边距。
- 响应式（桌面优先）：
  - Desktop（≥1024px）：面板默认右侧浮动（右上区域），宽高固定规格。
  - Tablet（768–1023px）：面板宽度按比例缩小（但仍与另一面板同规格）。
  - Mobile（≤767px）：面板改为底部抽屉（仍保持统计/标签一致规格），避免遮挡过多内容。

## 2) Meta Information（页面元信息）
- Title：Text play - 文本游玩
- Description：进行文本游玩并通过 📊 面板查看统计与标签信息。
- Open Graph：
  - og:title：Text play - 文本游玩
  - og:description：在同一页面通过 📊 菜单打开统计/标签浮动面板

## 3) Global Styles（全局样式/设计令牌建议）
- Background：#0B0F17（深色底）或沿用现有主题
- Surface（面板背景）：#111827（带轻微透明可选）
- Text：主文本 #E5E7EB；次级 #9CA3AF
- Accent：用于高亮与主按钮（沿用现有品牌色）
- Radius：面板/卡片 12px；小组件 8px
- Shadow：面板使用中等阴影（层级高于内容，低于全屏弹窗）
- Button：
  - 默认：透明/描边
  - Hover：仅对按钮本身（不触发展开），加强背景或描边
  - Active：轻微下压/色值加深
- Link：仅用于正文内；面板标题与按钮不使用链接样式，避免误导

## 4) Page Structure（页面结构）
1. 顶部工具条（TopBar）
2. 主内容区域（PlayContent）
3. 浮动层：
   - 📊 菜单（Popover/Menu）
   - 浮动面板（StatsPanel / TagsPanel）

## 5) Sections & Components（区块与组件详细说明）

### 5.1 顶部工具条（TopBar）
- 左侧：页面标题“Text play”（可带小型状态文本，如果现有页面已有）
- 右侧：
  - **📊 菜单按钮（IconButton）**
    - 点击：打开/关闭“面板选择菜单”
    - 键盘：
      - Tab 可聚焦
      - Enter/Space 打开
      - Esc 关闭菜单
    - 重要：不支持 hover 打开，不显示 hover 展开态

### 5.2 面板选择菜单（Panel Switcher Menu）
- 触发：仅来自 📊 菜单按钮点击/键盘激活
- 菜单项（最小集合）：
  1) 统计
  2) 标签
- 行为：
  - 点击“统计” -> 打开统计面板；若统计面板已开 -> 关闭
  - 点击“标签” -> 打开标签面板；若标签面板已开 -> 关闭
  - 若产品允许同屏多面板：可支持同时打开；若不允许：打开其一时关闭另一（在实现阶段二选一）
- 关闭：点击菜单外区域 / Esc / 再次点击 📊

### 5.3 浮动面板容器（PanelsHost）
- 职责：统一管理面板的定位、层级、关闭策略与尺寸规格。
- 定位（Desktop 默认）：
  - 右上区域：距离右侧 16px，距离 TopBar 下沿 12–16px
  - z-index：高于主内容与顶部栏内容层（但低于全屏模态层）
- 尺寸规格（关键）：
  - **StatsPanel 与 TagsPanel 采用同一份规格定义**：`width`、`height`、`maxHeight`、`padding`、`headerHeight` 完全一致
  - 内容溢出：面板 body 区内部滚动（`overflow: auto`），而不是改变面板外形

### 5.4 通用浮动面板壳（FloatingPanel）
- 结构：
  - Header：标题（“统计”/“标签”）+ 关闭按钮（X）
  - Body：可滚动内容容器
  - Footer（可选）：仅当当前页面已有必要操作（本次不新增）
- 交互：
  - 关闭：X / Esc / 点击面板外区域（按既有交互规范选择）
  - 聚焦：打开面板后将焦点移到 Header 或 Body 首元素；关闭后回焦到 📊 按钮

### 5.5 统计面板（Stats Panel）
- 目标：信息“直接可见”，不需要“more/更多”。
- 内容结构建议：
  - 使用列表或双列 Key-Value（按现有统计字段保持不变）
  - **移除“more/更多”入口与折叠区域**
  - 当条目较多：面板 Body 内滚动；可在顶部保留小型筛选/分组标题（仅当现有已存在）

### 5.6 标签面板（Tags Panel）
- 与统计面板保持一致规格与视觉层级。
- 内容结构：
  - 标签以列表或“可换行 chip”展示（按现有实现选择其一）
  - 溢出同样在面板 Body 内滚动

## 6) 交互状态与可用性细节
- 禁止 hover 展开：
  - 菜单与面板所有“打开动作”仅来自点击/键盘激活
  - hover 仅用于按钮/菜单项高亮，不触发展开
- 点击外部关闭：
  - 菜单与面板分别处理点击外部；避免“点菜单项导致立即关闭且不打开”的竞态
- 键盘路径：
  - Tab 到 📊 -> 打开菜单 -> 上下选择（可选）-> Enter 打开面板
  - Esc 优先关闭最上层（面板 > 菜单）
- 动效（可选，轻量）：
  - 菜单：淡入 + 轻微位移（80–120ms）
  - 面板：缩放/滑入（120–180ms），避免过度动画
